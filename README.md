See the source code for full details and usage instructions.

## Installation

### OpenClaw (recommended)

Copy the skill into your OpenClaw skills directory:

```bash
git clone https://github.com/NeoSkillFactory/ai-router-skill.git
cp -r ai-router-skill ~/.openclaw/skills/ai-router-skill
```

### Standalone

```bash
git clone https://github.com/NeoSkillFactory/ai-router-skill.git
cd ai-router-skill
npm install
```
